# 🚀 ISOLELE - Commencer en 2 Minutes

## Étape 1: Installer (30 secondes)
```bash
npm install
```

## Étape 2: Tester Gemini (20 secondes)
```bash
node test-gemini.js
```

Vous verrez:
```
✅ Text generation works!
✅ Translation works!
✅ Description generation works!
🎉 All tests passed!
```

## Étape 3: Lancer (10 secondes)
```bash
npm run dev
```

## Étape 4: Ouvrir dans le navigateur
```
http://localhost:3000
```
