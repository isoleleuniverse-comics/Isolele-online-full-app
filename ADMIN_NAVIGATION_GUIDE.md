# Guide d'Navigation Admin ISOLELE

## Accès à l'Admin

**URL:** `/admin` ou `https://votre-site.com/admin`

---

## Structure du Sidebar

Le sidebar gauche contient le menu principal avec les icônes suivantes:

### 1️⃣ **Tableau de Bord** (Dashboard)
- **URL:** `/admin`
- **Contenu:**
  - ISOLELE Brand Section avec logo
  - Brand Assets Gallery (galerie d'images)
  - Stats Cards (Visites, Commandes, Articles, Revenus)
  - Traffic Chart
  - Notifications
- **Icône:** 📊 (Dashboard)

### 2️⃣ **Contrôle du Site** ⭐ NOUVEAU
- **URL:** `/admin/site-controller`
- **Contenu:**
  - Paramètres du Site (nom, description, couleurs)
  - Gestion des Pages (publier/dépublier)
  - Mode Maintenance ON/OFF
  - Activation des features (Analytics, Newsletter, Commentaires)
- **Icône:** ⚙️ (Settings)
- **CECI EST LA PAGE PRINCIPALE POUR CONTRÔLER LE SITE PUBLIC**

### 3️⃣ **Actualités**
- **URL:** `/admin/news`
- **Contenu:** Gestion des articles d'actualités
- **Icône:** 📄 (FileText)

### 4️⃣ **Personnages**
- **URL:** `/admin/characters`
- **Contenu:** Gestion des personnages ISOLELE
- **Icône:** 👥 (Users)

### 5️⃣ **Boutique**
- **URL:** `/admin/shop`
- **Contenu:** Gestion des produits et commandes
- **Icône:** 🛍️ (ShoppingBag)

### 6️⃣ **Médias**
- **URL:** `/admin/media`
- **Contenu:** Galerie d'images et uploads
- **Icône:** 🖼️ (Image)

### 7️⃣ **Pages**
- **URL:** `/admin/pages`
- **Contenu:** Gestion des pages publiques
- **Icône:** 📖 (BookOpen)

### 8️⃣ **Analytics**
- **URL:** `/admin/analytics`
- **Contenu:** Statistiques de trafic
- **Icône:** 📈 (BarChart)

### 9️⃣ **Sécurité**
- **URL:** `/admin/security`
- **Contenu:** Paramètres de sécurité
- **Icône:** 🔒 (Shield)

### 🔟 **Paramètres**
- **URL:** `/admin/settings`
- **Contenu:** Paramètres globaux
- **Icône:** ⚙️ (Settings)

## Comment Naviguer

### Desktop (Sidebar Visible)
1. Le sidebar est visible à gauche
2. Cliquez sur un élément pour naviguer
3. Utilisez le bouton < à gauche du logo ISOLELE pour collapser le sidebar
4. Les icônes restent visibles même quand collapsé

### Mobile (Menu Hamburger)
1. Cliquez le bouton ☰ en haut à gauche
2. Un menu slide-out apparaît depuis la gauche
3. Tapez sur un élément pour naviguer
4. Le menu se ferme automatiquement

### Avec Clavier
- Les liens sont accessibles via Tab
- Appuyez Enter pour naviguer

## Feature Spotlight: Contrôle du Site

La page **Contrôle du Site** est la plus importante. Elle vous permet de:

### 1. Gérer les Paramètres du Site
```
Nom du Site: ISOLELE
Description: African Heritage & Mythology Universe
Couleur Primaire: #D4AF37
Couleur Secondaire: #B3541E
Couleur Accent: #C9A542
```

### 2. Gérer les Pages Publiques
Pour chaque page, vous pouvez:
- **Publier/Dépublier** avec un clic (rend visible/invisible aux visiteurs)
- **Afficher/Masquer** rapidement
- **Éditer les métadonnées** (titre, description)
- **Marquer comme Featured** pour la homepage

### 3. Activer/Désactiver les Features
- 📊 **Analytics:** Suivi des visiteurs avec Google Analytics
- 📧 **Newsletter:** Collecte d'emails des abonnés
- 💬 **Commentaires:** Activez les commentaires sur les articles

### 4. Mode Maintenance
Mettez le site en maintenance d'un clic pour:
- Faire des mises à jour sans visiteurs
- Maintenance technique
- Changements de design

## Statut Visuel

### Indicateurs de Pages
- 🟢 **Vert:** Page publique et visible
- 🔴 **Rouge:** Page privée
- ⚫ **Gris:** Page cachée

### Indicateurs de Status
- ✅ **Checkmark vert:** Sauvegardé avec succès
- ⏳ **Spinner:** En cours de sauvegarde
- ❌ **X rouge:** Erreur lors de la sauvegarde

---

## Notifications

### Types de Notifications
- ✅ Succès (page sauvegardée, paramètres appliqués)
- ⚠️ Avertissement (changement important)
- ❌ Erreur (problème de connexion, validation)
- ℹ️ Info (nouveau contenu, mise à jour)

### Notifications Auto-Dismiss
Les notifications disparaissent automatiquement après 3-5 secondes sauf:
- Les erreurs restent affichées
- Vous pouvez les fermer manuellement

## Conseils d'Utilisation

1. **Avant de faire un changement:** Prenez une screenshot
2. **Après un changement:** Allez vérifier sur le site public
3. **Régulièrement:** Téléchargez les analytics
4. **Avant les mises à jour:** Activez le mode maintenance
5. **Utilisez les raccourcis** pour naviguer plus vite
