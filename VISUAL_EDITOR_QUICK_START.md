# Visual Editor - Quick Start (5 Minutes)

## 🚀 Get Started Right Now

### Step 1: Open the Editor (2 seconds)
```
Go to: /admin/home/refact
Click: "Display Refact (Live Preview)" tab
```

You'll see:
- Your entire public website in the middle
- A draggable control panel (bottom right)
- Professional dark theme with gold accents

### Step 2: Switch to Edit Mode (1 second)
In the control panel, click the **purple "Edit" button**

Status should now say: **"Edit mode active"**

### Step 3: Click Something to Edit (5 seconds)
Click ANY text or image in your website preview

You'll see:
- Golden outline appears around the element ✨
- Glass-morphism input box shows up
- Current content is selected

### Step 4: Type Your Changes (10 seconds)
Start typing your new content right in the input box

Example:
```
Old: "Welcome to ISOLELE"
New: "Welcome to ISOLELE Universe"
```

### Step 5: Save the Edit (1 second)
Click the **green checkmark button** on the right side of the input

Result: Your element updates INSTANTLY in the preview!

### Step 6: Save Everything & Go Live (30 seconds)
Click the big **green "Save & Deploy" button** in the control panel

You'll see:
- Progress modal appears
- Three phases:
  1. Collecting changes (0-20%)
  2. Syncing to GitHub (20-50%)
  3. Deploying to Vercel (50-100%)
- Success message: "Changes deployed successfully!"
- **New tab opens with your live changes**

**Done! Everyone can see your changes!** 🎉

## 💡 Pro Tips

**Tip 1: Move the Panel**
- Grab the dark header of the panel
- Drag it anywhere on screen
- Use to see content behind the panel

**Tip 2: Keyboard Shortcuts**
- While in input box:
  - `Enter` = Save
  - `Escape` = Cancel
  - `Cmd/Ctrl + A` = Select all

**Tip 3: Undo**
- Before clicking Save & Deploy
- Click X to close input
- Element reverts to original

**Tip 4: Check Live Site**
- Click the external link icon (top right)
- Opens your live public site
- Compare with preview

**Tip 5: Multiple Edits**
- Make as many changes as you want
- Click checkmark after each one
- Click Save & Deploy once at the end

## ⚠️ Common Mistakes (and fixes)

### "I clicked Edit but nothing happens"
✅ **Fix:** Make sure you're clicking on actual content, not empty space

### "The input box disappeared"
✅ **Fix:** It's probably behind the control panel. Drag the panel to the side.

### "Changes aren't saving"
✅ **Fix:** Check you have GitHub and Vercel tokens set up. See SETUP_AUTO_DEPLOYMENT.md

### "Save button is greyed out"
✅ **Fix:** Finish your current edit first. Click checkmark or press Escape.
