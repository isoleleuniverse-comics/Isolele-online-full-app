## The 4 Editing Modes

### 🖐️ Hand Mode
Navigate your site. Click elements to select them.

### ✏️ Edit Mode
Modify the selected element. Change text, colors, spacing.

### ✨ Animation Mode
AI suggests animations. Pick your favorite. It's added.

### 📱 Media Mode
Upload new images. Replace existing ones. Simple.
