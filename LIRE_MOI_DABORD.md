## 🐛 SI QUELQUE CHOSE NE FONCTIONNE PAS

### Les images ne s'affichent pas?
- Vérifiez votre connexion internet
- Videz le cache (Ctrl+Shift+Del)
- Rafraîchissez (Ctrl+F5)

### Le chatbot ne répond pas?
- Vérifiez que `.env.local` existe
- Vérifiez que la clé API est correcte
- Ouvrez Console (F12) pour voir les erreurs

### La nav mobile n'apparaît pas?
- Ouvrez DevTools (F12)
- Allez dans "Toggle device toolbar"
- Sélectionnez une taille mobile (< 1024px)
- Rafraîchissez la page

### Rien ne fonctionne?
1. Videz complètement le cache
2. Fermez tous les onglets du projet
3. Relancez `npm install` et `npm run dev`
4. Attendez 30 secondes que tout compile
